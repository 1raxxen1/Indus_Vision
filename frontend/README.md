# Frontend (React) Skeleton

This folder is a starter location for your React frontend app.

## Suggested next steps
1. Initialize React app here (Vite recommended):
   - `npm create vite@latest . -- --template react`
2. Install dependencies:
   - `npm install`
3. Use `src/api/endpoints.js` to keep backend route constants in one place.

## Backend base URL
Default Django local URL: `http://127.0.0.1:8000`
